# UpstreamWatchr

Set ENV['GITLAB_API_ENDPOINT'] and ENV['GITLAB_API_PRIVATE_TOKEN'] and call the bin with your projects url.

## Installation

Add this line to your application's Gemfile:

    gem 'upstreamwatchr'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install upstreamwatchr
